💻 OS Deadlock Simulator & Visualization

An interactive Operating System project designed to simulate, visualize, and analyze deadlocks, resource allocation, and process synchronization concepts in a dynamic and user-friendly way.

🚀 Project Overview

This project helps in understanding core OS concepts like:

Deadlock Detection
Deadlock Prevention
Banker's Algorithm
Wait-For Graph
Resource Allocation

It provides a real-time visual simulation of processes and resources, making complex concepts easier to grasp.

🧠 Key Features

✅ Deadlock Detection System

Identifies deadlock situations instantly
Highlights affected processes

✅ Banker’s Algorithm Implementation

Checks system safety
Displays safe sequence (if exists)

✅ Wait-For Graph Visualization

Graphical representation of process dependencies
Easy identification of circular wait

✅ Interactive Resource Allocation

Add/remove resources dynamically
Modify process needs in real time

✅ System State Monitoring

Safe / Unsafe state indication
Real-time updates

✅ Scenario Library

Predefined cases:
Beginner (Simple Deadlock)
Intermediate (Safe State)
Advanced (Circular Wait)
📊 System Components
🔹 Processes & Resources
Multiple processes (P0, P1, P2, …)
Resource types (A, B, C or R0, R1, R2, …)
🔹 Matrices Used
Allocation Matrix
Maximum Matrix
Need Matrix
Available Resources
⚙️ Algorithms Used
🔸 Banker’s Algorithm
Ensures system remains in a safe state
Calculates safe execution sequence
🔸 Deadlock Detection Algorithm
Detects circular wait conditions
Identifies blocked processes
🔸 Wait-For Graph Algorithm
Converts resource allocation graph
Detects cycles for deadlock
⚠️ Deadlock Conditions (Coffman Conditions)

Deadlock occurs if all four conditions hold:

Mutual Exclusion
Hold and Wait
No Preemption
Circular Wait
